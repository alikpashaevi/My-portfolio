Simon Game

The Simon game is a classic electronic memory game where players must repeat a sequence of lights and sounds generated by the game. The game starts with one light/sound, and each successful repetition adds another to the sequence. Players continue to repeat the sequence until they make a mistake, at which point the game ends. It challenges memory and reflexes, often played competitively or for fun.
